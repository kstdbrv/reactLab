SIR the Label


Разработка интернет-магазина для международного производителя одежды.

us.sirthelabel.com задачи

- Запустить цифровой канал продаж производителя для розничных и оптовых покупателей;
- Расширить В2В-аудиторию за счет малого бизнеса и конечных бизнес-покупателей;
- Наладить прямой диалог с розничными покупателями;

- Интегрировать интернет-магазин с фулфилмент-сервисом;
- Упростить процесс покупки и документооборот для B2B-клиентов;
- Обеспечить повышенную безопасность обмена и хранения данных;
- Автоматизировать определение стоимости доставки
- Обеспечить возможность отслеживания статуса заказа;

решение

Друзья и соучредители Никки Кэмпбелл и Софи Кут запустили SIR, когда им было 24 года. Они разделяют любовь к кодам стиля 1970-х годов - среди муз входят Франсуаза Харди и Пэт Кливленд - которые влияют на каждую из их коллекций.

  1. отображение разных цен для розничных и нескольких категорий B2B-покупателей;
  2. разделение закупок по филиалам и дочерним компаниям в рамках одного аккаунта;
  3. система уведомлений в личном кабинете B2B-покупателя, в том числе о просроченной задолженности;
  4. использование системы обмена сообщениями RabbitMQ для оперативной обработки запросов;
  5. возможность отслеживать статус заказа и получать актуальные документы на каждом из этапов;
  6. автоматическое формирование документов, включая заявки на отгрузку;
  7. обмен данными через промежуточную базу данных для обеспечения безопасности;
  8. хранение коммерческих данных по сессиям и с настроенным шифрованием для безопасности данных.
